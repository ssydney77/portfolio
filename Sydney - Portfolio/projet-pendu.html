Projet : Pendu

Projet réalisé en Python

Cahier des charges

● Créer le jeu du pendu en langage Python

● Définir une fonction "Dictionnaire" qui contiendra la liste des mots possibles

● Proposer un mot aléatoire grâce à la fonction "Dictionnaire".

● Par rapport au mot aléatoire choisi :

- le joueur devra indiquez une lettre dans le champ de saisie.

- selon la lettre choisie et le mot aléatoire défini pour cette manche, il faut indiquer au joueur quelles sont la/les lettre(s) bien placée(s) s'il y en a. Sinon, afficher le nombre d'essais restant.

Conditions de victoire

Pour que le joueur gagne au pendu, il y a 2 manières possibles :

1. Si le joueur trouve toutes les lettres du mot à chercher il gagne, même si il n'a pas tenté de l'écrire

2. Si le joueur tente un mot et qu'il s'avère être le bon, il gagne le jeu

Le code source

import random
          
# Liste des mots "secrets" possibles
          
dictionnaire=["chauve","monument","insecte","tornade","photocopie","chapeau","champagne","magasin",
              "agrafeuse","pharmacie","observatoire","zoo","janvier","princesse","honte","novice",  
              "population","xylophone","gazelle","moteur","sentiment","dromadaire","appartement",
              "filtre","barbe","plastique","vivant","casserole","jus","assassin","grand","table"]

# Fonction sélectionnant un mot aléatoire dans la liste dictionnaire

def mot(dictionnaire):
  aleatoire=random.randint(0,len(dictionnaire)-1)
  mot_a_trouver=dictionnaire[aleatoire]
  return mot_a_trouver
  
# Initialisation des variables
              
mot_a_trouver=(mot(dictionnaire))
essai=7
mot_code=''

# Création d'une chaine de caractères indiquant le nombre de lettres du mot secret
              
for i in range (len(mot_a_trouver)):
mot_code=mot_code+"_"
print(mot_code)

# Création d'une boucle fonctionnant tant que le mot codé n'est pas équivalent au mot secret et que le nombre d'essai est supérieur à 0

while mot_code != mot_a_trouver and essai>0:

  # Proposer à l'utilisateur s'il souhaite ou non saisir le mot secret
  o_n=str(input("\nVoulez vous tenter de saisir le mot? (o/n) "))

  # Si l'utilisateur décide : non
  if o_n=="n":                                                       
  lettre=str(input("\nVeuillez saisir une lettre: "))
  essai=essai-1
  verif=False

    # Boucle permettant d'afficher la/les lettre(s) choisie(s) par l'utilisateur, à l'emplacement dans le mot codé
    for i in range(len(mot_a_trouver)):
      if(mot_a_trouver[i]==lettre):
        mot_liste = list(mot_code)
        mot_liste[i] = lettre
        mot_code = ''.join(mot_liste)
        verif=True

      # Si la lettre saisie par l'utilisateur n'est pas dans le mot secret
      if verif==False:
        print("\nDommage votre lettre n'est pas dans le mot")
        print("\nIl vous reste encore",essai,"essais.\n")
        print(mot_code)
                  
      # Sinon
      else:
        essai=essai+1
        print("\nSuper votre lettre est dans le mot\n")
        print("\nIl vous reste encore",essai,"essais.\n")
        print(mot_code)
                
  # Si l'utilisateur décide : oui
  if o_n=="o":
    user_mot=str(input("\nVeuillez saisir le mot: "))
    
    # Si le mot saisi par l'utilisateur est identique au mot secret
    if (user_mot==mot_a_trouver):
      print("\nBravo vous avez trouver le mot !")
      break;
    
    # Sinon
    else:
        print("\nDommage vous n'avez pas trouver le mot, il vous reste donc",essai-1,"essais !")
        essai=essai-1
        print(mot_code)
              
# Si l'utilisateur trouve le mot secret uniquement avec des lettres saisies 
if mot_code == mot_a_trouver:
  print("\nBravo vous avez trouver le mot !")
              
# Si le nombre d'essais est égal à 0
if essai==0:
print("\nVous avez perdu ! le mot a trouver été :",mot_a_trouver)
